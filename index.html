Vim Commands

Basic Movement

h Move left
j Move down
k Move up
l Move right

Line Movement

0 Go to first column
$ Go to last column

Word Movement

w Next obstacle/item
b Previous obstacle/item
e End of next item
ge End of previous item

Screen Movement

gg Go to top (safe row)
G Go to bottom (safe row)
Ctrl+f Page down
Ctrl+b Page up

Count Prefix

5j Move down 5 times
3l Move right 3 times

Add any number before a movement!
